# GlossAPI Documentation

Welcome to the GlossAPI docs. GlossAPI is a GPU‑accelerated pipeline for processing academic documents (with a focus on Greek), including download → layout/OCR → cleaning → sectioning → classification, and optional math/code enrichment.

- Getting Started: getting_started.md
- Quickstart: quickstart.md
- Pipeline Overview & Artifacts: pipeline.md
- OCR and Math Enrichment: ocr_and_math_enhancement.md
- Multi‑GPU & Benchmarking: multi_gpu.md
- Configuration (env vars & knobs): configuration.md
- API Reference — Corpus: api/corpus.md
- Troubleshooting: troubleshooting.md

## At a Glance

- Install: `pip install glossapi`
- Run GPU OCR: see quickstart.md
- Run Phase‑2 math enrichment from JSON: see ocr_and_math_enhancement.md

For code examples, production tips, and folder structure details, see the links above.
