### EELLAK的目标
EELLAK的目标是开发一个带有开源代码的广元语言模型，名举glossAPI（一个广元开源语言模型）。该模型的代码将在EUPL许可下揭优化，并通过Creative Commons BY-SA许可分享作为数据。
